# Zautomatyzuj Się

Statyczna strona HTML przygotowana do publikacji przez GitHub Pages.

## Struktura

- `index.html` - główny plik strony
- `assets/logo-z.png` - logo używane przez stronę

## Publikacja

Repozytorium jest gotowe do wypchnięcia na GitHub i publikacji przez GitHub Pages z gałęzi `main`.
